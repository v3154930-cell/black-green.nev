# Project Summary

## Black Green New

Next.js storefront for Black Green tea shop brand.

### Tech Stack
- Next.js 16.2.0
- TypeScript
- Tailwind CSS 3.4
- React 19

### Status: Development

### Routes
- / - Main storefront (13 blocks implemented)
- /catalog - Product catalog
- /catalog/[category] - Category page
- /product/[slug] - Product detail
- /about - About page
- /contacts - Contacts page
- /delivery - Delivery info
- /news - News listing
- /news/[slug] - News detail
- /reviews - Reviews listing
- /admin - Admin overview
- /admin/moderation - Moderation queue
- /admin/content - Content drafts

### Data
Mock data in src/lib/data.ts (products, categories, news, reviews)

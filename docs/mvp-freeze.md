# MVP Freeze

Главная страница приведена к минимально жизнеспособному состоянию.

## Что включено

1. **Hero** — главный баннер с time-of-day логикой
2. **Каталог** — 4 категории (tea, tea-drinks, teaware, gifts)
3. **Хиты** — 3 товара
4. **Подарки** — готовые наборы
5. **Отзывы** — живые отклики
6. **Доставка и сервис** — Москва/регионы
7. **Footer** — навигация

## Принятые контентные решения

### Журнал
- Журнал принят как контентный слой (`/news`)
- Формат: новости, обновления ассортимента, сезонные подборки
- Статус: в работе

### MAX как главный канал связи
- MAX указан как основной канал в TopBar
- Telegram — вторичный канал
- Реализовано в `src/components/TopBar.tsx`

### Формат изображений 3:4
- Все товарные карточки используют aspect-ratio `3:4`
- Реализовано в ProductCard и на странице товара
- FallbackImage автоматически подменяет отсутствующие изображения

## Что отложено

- Скидки (отдельный маршрут)
- Преимущества (можно на /about)
- Поддержка (Telegram/email)

## Search Layer (Sprint 1)

- Базовый поиск по каталогу реализован
- Поиск по названию, категории, ключевым словам
- Пустая выдача (empty state) с подсказками
- Подготовлено для future voice input
- Документация: `docs/search-layer.md`

## Компоненты

Переиспользуемые:
- `Hero`, `CategoryCard`, `ProductCard`, `FallbackImage`
- `TopBar`, `Header`
- `SearchInput`, `SearchResults`, `CatalogSearch` (новые)

## Fallback для изображений

Все карточки используют `FallbackImage`, который автоматически подменяет отсутствующие изображения на `/images/placeholder.svg`.

## Usability каталога

- Breadcrumbs: главная → каталог → категория → товар
- "Назад в каталог" на странице товара
- Активные состояния категорий в навигации
- Активная ссылка в Header подсвечена зелёным
